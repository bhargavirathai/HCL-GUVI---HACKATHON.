# Financial Sales Analysis using Python

This project analyzes financial sales data using Python and Pandas.

## Features
- Data Cleaning
- Profit Analysis
- Country-wise Profit
- Product-wise Profit
- Segment-wise Profit
- Visualizations using Matplotlib

## Tech Stack
Python, Pandas, Matplotlib, OpenPyXL

## Results
Total Sales: 118,726,350
Total Profit: 16,893,702
